# Eido

An AI agent platform for diverse scenarios. Powered by Claude Code SDK, it autonomously plans and executes complex tasks with multi-skill orchestration and multi-turn collaboration.

## Features

- **Autonomous execution**: Based on Claude Code SDK, the AI autonomously breaks down tasks, invokes tools, and iterates until completion
- **Skill system**: Define skills via SKILL.md files with tool allowlists and natural-language guidance—extend without coding
- **Multi-LLM support**: Compatible with DeepSeek, MiniMax, and other Anthropic API-compatible services
- **Ready to deploy**: One-command Docker deployment, supports amd64 / arm64

See [quick-start.md](quick-start.md) for details.
