# 申万二级行业行情分析技能

## 技能概述

本技能用于分析申万二级行业的行情变化，自动获取全部124个申万二级行业的实时行情数据，生成专业的HTML。

**纯AKShare实现**

## 使用场景

- 用户请求分析申万二级行业行情
- 用户需要生成行业分析报告

## 执行流程

### 手动执行

```bash
python3 sw-industry-report/scripts/fetch_and_send.py
```


## 核心脚本

**fetch_and_send.py**: 完整执行脚本，包含以下功能：

1. **获取行业基本信息**：使用 `ak.sw_index_second_info()` 获取申万二级行业基本信息（名称、上级行业、PE、PB等）
2. **获取实时行情**：使用 `ak.index_realtime_sw(symbol='二级行业')` 获取实时行情
3. **生成HTML报告**：包含涨跌幅排名、统计概览、全部行业分布

## 数据来源

- **行业信息**：AKShare `sw_index_second_info()` → 申万宏源研究
- **实时行情**：AKShare `index_realtime_sw(symbol='二级行业')` → 申万宏源研究

## 报告内容

生成的HTML报告包含：
1. **数据概览**：行业总数、上涨/下跌/平盘数量
2. **涨幅前20行业**：按涨跌幅降序排列，显示成交量和成交额
3. **跌幅前20行业**：按涨跌幅升序排列
4. **全部行业涨跌分布**：卡片式展示所有行业
5. **配色说明**：涨幅（红色）、跌幅（绿色）
