## 技能执行流水线

```mermaid
sequenceDiagram
    actor 用户
    participant 前端 as 前端 ChatArea
    participant 后端 as FastAPI /chat/chat
    participant SVC as ClaudeSkillService
    participant SDK as claude_agent_sdk
    participant 文件 as 本地文件系统

    用户->>前端: 输入 "@技能A ... @技能B ..."
    前端->>前端: 解析 @提及，按顺序排列 [技能A, 技能B]

    Note over 前端: 多技能流水线开始

    前端->>后端: POST /chat/chat {skill_id: A, messages}
    后端->>SVC: execute_stream(skill_id_A, messages)
    SVC->>SVC: 读取 SKILL.md\n格式化对话历史\n构建 Prompt
    SVC->>SDK: query(prompt, allowed_tools)
    SDK-->>前端: SSE: thinking / tool_use / content ...
    SDK->>文件: Read / Write / Bash ...
    SDK-->>SVC: 流结束
    SVC-->>前端: SSE: workflow_complete

    Note over 前端: 技能A 完成，output_A 作为 context 传入技能B

    前端->>后端: POST /chat/chat {skill_id: B, messages, context: output_A}
    后端->>SVC: execute_stream(skill_id_B, messages, context)
    SVC->>SVC: Prompt 追加\n"上一步执行结果: output_A"
    SVC->>SDK: query(prompt, allowed_tools)
    SDK-->>前端: SSE: thinking / content ...
    SVC-->>前端: SSE: workflow_complete

    前端->>用户: 渲染两条 assistant 消息
```
